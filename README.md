# Inventory & Order Management System

Backend inventory management system built using Spring Boot.

## Features
- Product and warehouse management
- Inventory tracking per warehouse
- Transactional stock add/remove operations
- Validation to prevent overselling
- REST APIs tested using Postman

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- H2 Database
